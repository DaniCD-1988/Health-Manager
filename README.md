# HEALTH MANAGER - v1.2.0

**Nota:** Este repositorio pertenece a un proyecto académico de fin de ciclo formativo, no es una aplicación comercial en producción.

## Descripción:
**Health Manager** es una aplicación móvil nativa para Android diseñada para ayudar a los usuarios a adoptar un estilo de vida más saludable al mismo tiempo que gestionan su productividad diaria. 

El proyecto fue desarrollado de forma colaborativa como **Proyecto Final Intermodular** para el CFGS de Desarrollo de Aplicaciones Multiplataforma (DAM).

## Características Principales:
*   **Registro de datos biométricos y métricas personales:** Seguimiento de la evolución física del usuario.
*   **Sección de bienestar:** Consejos (tips) de salud y retos diarios motivacionales.
*   **Módulo de productividad:** Calendario dinámico con gestión de eventos y recordatorios.
*   **Módulo de actividad:** Cronómetro integrado para contabilizar el tiempo de ejercicio físico.

## Tecnologías Utilizadas:
*   **Lenguaje:** Java.
*   **Entorno de Desarrollo (IDE):** Android Studio.
*   **Base de Datos:** SQLite (Persistencia de datos local estructurada).
*   **Control de Versiones:** Git y GitHub.
